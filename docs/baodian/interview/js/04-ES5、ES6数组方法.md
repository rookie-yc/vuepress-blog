## ES5、ES6数组方法

### ES5系列

#### indexOf
**用途**： 用于查找数组中是否存在某个值，如果存在则返回某个值的下标，否则返回-1

- 第一个参数，查找的元素
- 第二个参数，可选，规定在数组中开始检索的位置。它的合法取值是 0 到 stringObject.length - 1。如省略该参数，则将从字符串的首字符开始检索。

```js
let list = [1, 2, 3];

console.log(list.indexOf(2)) // 1
console.log(list.indexOf("蛙人")) // -1
```
